Funtions

Function Statement

function nameFunction() { code here }

Function Expression

Or anonymous function: const nameFunction = function(parameters) { code here }

Function Scope

Be sure, to use parameters and variables in functions correctly!

Function Hoisting

NOTE: If you want to run a function and create it later, you can't use a function Expression. You just can use a function statement.

Arrow Function

To use a arrow function, do this: const myArrowFunction = (parameters) => { code here }

Callback Function

Example of a callback function:

function myCallback(cb) {
        // console.log(cb);
        cb();
    }
    myCallback(
        () => {
            console.log("I'm in a Callback");
        }
    );

Repetition Structures

For

Code exemple:

for (let i = 100; i >= 0; i -= 10) {
    console.log(i);
}

While

Code exemple:

let i = 0;
while (i < 10) {
    console.log("While ... " + i);
    i++;
}

For...of

Code exemple:

let myName = "Rayan";
for (let char of myName) {
    console.log(char);
}

For...of

Use it to display object values/properties. Code exemple:

for (let property in person) {
    console.log(property);
}

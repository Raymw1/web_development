Learning CSS - RocketSeat

The meaning of CSS is: Cascading Style Sheet. We use it to create styles to our HTML.

How can you select elements?

  • The name of the element
  • IDs
  • Classes
  • Pseudoclasses, attributes

You can style everything in the pege using * { /*styles*/ }

Learning box-model in this page

How can you use CSS?

  • Inline(style attribute, like this)
  • Style TAG
  • Link TAG(external css)
  • @import(external css)

The order of cascading

  1. Style origin (inline > tag style > tag link) - by last
  2. Specificity (Universal < Element type and pseudo < class and attribute < IDs < Inline)
  3. !important rule(AVOID)

At rules

  • @import(External CSS)
  • @media(Responsive)
  • @font-face(External Fonts)
  • @keyframes(animations)
